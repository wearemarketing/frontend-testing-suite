# frontend-testing-suite
> Powered with GULP, postCSS, Browserify, Jest..


#### Run tests

```sh
$ npm test
```

#### Run tests with _watch_
```sh
$ npm test -- --watchAll
```

#### Run tests with _coverage_
```sh
$ npm test -- --watchAll --coverage
```
